def molSimplify.Scripts.structgen.ffopt (   ff,

  constopt,

  frozenangles,

  nsteps,
  debug = False 
)

Main constrained FF opt routine.

To optimize metal-containing complexes with MMFF94, an intricate procedure of masking the metal atoms and manually editing their valences is applied.

OpenBabel's implementation of MMFF94 may run extremely slowly on some systems. If so, consider switching to UFF.

Parameters
ffForce field to use, available MMFF94, UFF, Ghemical, GAFF
molmol3D of molecule to be optimized
connectedList of indices of connection atoms to metal
constoptFlag for constrained optimization - 0: unconstrained, 1: fixed connecting atom positions, 2: fixed connecting atom distances
frozenatsList of frozen atom indices
frozenanglesFlag for frozen angles, equivalent to constopt==1
mlbondsList of M-L bonds for distance constraints
nstepsNumber of steps to take - Adaptive: run only enough steps to remove clashes, default 200
debugFlag for debug info printing
Returns
FF-calculated energy, mol3D of optimized molecule
